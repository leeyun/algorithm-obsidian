## 1. 덱에 대한 설명

덱은 양쪽 끝에서 데이터의 삽입과 삭제가 모두 가능한 선형 자료구조다. 스택과 큐의 특징을 혼합한 형태를 띤다.

* 양방향 구조: 데이터가 들어오고 나가는 입구가 양쪽 끝에 모두 존재하여 앞이나 뒤 어느 방향에서든 접근할 수 있다.
* 주요 연산: 데이터를 앞에 넣는 연산을 push_front, 뒤에 넣는 연산을 push_back이라고 한다. 앞의 데이터를 꺼내는 연산을 pop_front, 뒤의 데이터를 꺼내는 연산을 pop_back이라고 한다.
* 장점: 큐와 스택의 장점을 모두 가지고 있어 상황에 맞게 데이터를 유연하게 처리할 수 있다. 양쪽 끝에서의 삽입과 삭제가 독립적으로 이루어지므로 연산 속도가 매우 빠르다.
* 단점: 단순한 스택이나 큐에 비해 내부 구현이 다소 까다롭다. 크기가 고정된 배열로 구현할 경우 원형 구조를 다뤄야 하므로 인덱스를 계산하는 로직이 복잡해진다. 중간에 위치한 데이터에는 직접 접근하거나 검색하기 어렵다.

## 2. 의사코드

원형 배열을 기반으로 덱을 구현했을 때의 핵심 연산에 대한 기본 논리를 보여주는 의사코드다.

```text
// 1. 앞에 데이터 삽입
function push_front(deque, value, front, max_size):
    if 덱 공간이 가득 찼다면:
        return "공간 가득 참 오류"
    
    // 앞쪽 포인터를 한 칸 앞으로 이동 (원형 구조 고려)
    front = (front - 1 + max_size) % max_size
    deque[front] = value

// 2. 뒤에 데이터 삽입
function push_back(deque, value, rear, max_size):
    if 덱 공간이 가득 찼다면:
        return "공간 가득 참 오류"
    
    // 뒤쪽 포인터를 한 칸 뒤로 이동 (원형 구조 고려)
    rear = (rear + 1) % max_size
    deque[rear] = value

// 3. 앞에서 데이터 삭제 및 반환
function pop_front(deque, front, max_size):
    if 덱이 비어있다면:
        return "비어있음 오류"
    
    value = deque[front]
    deque[front] = null
    front = (front + 1) % max_size
    return value

// 4. 뒤에서 데이터 삭제 및 반환
function pop_back(deque, rear, max_size):
    if 덱이 비어있다면:
        return "비어있음 오류"
    
    value = deque[rear]
    deque[rear] = null
    rear = (rear - 1 + max_size) % max_size
    return value
```

## 3. 시간 복잡도

덱은 양쪽 끝에서만 삽입과 삭제가 일어나므로 배열 이동이 필요 없어 매우 효율적이다.

| 연산 | 시간 복잡도 | 설명 |
| :--- | :--- | :--- |
| 앞에 삽입 | O(1) | 맨 앞에 데이터를 바로 추가하므로 다른 요소를 이동시킬 필요가 없다. |
| 뒤에 삽입 | O(1) | 맨 뒤에 데이터를 바로 추가하므로 다른 요소를 이동시킬 필요가 없다. |
| 앞에서 삭제 | O(1) | 맨 앞 데이터를 바로 제거하므로 다른 요소를 이동시킬 필요가 없다. |
| 뒤에서 삭제 | O(1) | 맨 뒤 데이터를 바로 제거하므로 다른 요소를 이동시킬 필요가 없다. |
| 검색 | O(n) | 특정 데이터를 찾기 위해 앞이나 뒤부터 순차적으로 하나씩 꺼내며 탐색해야 한다. |

## 4. C 언어 및 파이썬 예시

### C 언어

C 언어에서는 배열과 앞뒤 인덱스를 관리하는 변수를 통해 원형 덱을 구현한다.

```c
#include <stdio.h>
#define MAX_SIZE 5

int deque[MAX_SIZE];
int front = 0;
int rear = 0;

// 덱이 꽉 찼는지 확인
int is_full() {
    return (rear + 1) % MAX_SIZE == front;
}

// 덱이 비어있는지 확인
int is_empty() {
    return front == rear;
}

// 뒤에 데이터 삽입
void push_back(int value) {
    if (is_full()) {
        printf("덱 공간 가득 참\n");
        return;
    }
    rear = (rear + 1) % MAX_SIZE;
    deque[rear] = value;
}

// 앞에 데이터 삽입
void push_front(int value) {
    if (is_full()) {
        printf("덱 공간 가득 참\n");
        return;
    }
    deque[front] = value;
    front = (front - 1 + MAX_SIZE) % MAX_SIZE;
}

// 앞에서 데이터 삭제 및 반환
int pop_front() {
    if (is_empty()) {
        printf("덱 비어있음\n");
        return -1;
    }
    front = (front + 1) % MAX_SIZE;
    return deque[front];
}

// 뒤에서 데이터 삭제 및 반환
int pop_back() {
    if (is_empty()) {
        printf("덱 비어있음\n");
        return -1;
    }
    int value = deque[rear];
    rear = (rear - 1 + MAX_SIZE) % MAX_SIZE;
    return value;
}

int main() {
    push_back(10);
    push_front(20);
    push_back(30);

    printf("앞에서 pop된 값: %d\n", pop_front());
    printf("뒤에서 pop된 값: %d\n", pop_back());

    return 0;
}
```

### 파이썬

파이썬에서는 기본 내장 모듈인 collections의 deque를 사용하여 매우 직관적이고 강력한 덱 연산을 수행할 수 있다.

```python
from collections import deque

# 덱 초기화
dq = deque()

# 뒤에 데이터 삽입
dq.append(10)
dq.append(20)

# 앞에 데이터 삽입
dq.appendleft(30)
dq.appendleft(40)

# 앞에서 데이터 삭제 및 반환
print(f"앞에서 pop된 값: {dq.popleft()}")

# 뒤에서 데이터 삭제 및 반환
print(f"뒤에서 pop된 값: {dq.pop()}")

# 현재 덱 상태 확인
print(f"남은 덱: {list(dq)}")
```
