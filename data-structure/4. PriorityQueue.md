## 1. 우선순위 큐에 대한 설명

우선순위 큐는 들어온 순서와 상관없이 우선순위가 높은 데이터가 먼저 삭제되는 자료구조다. 내부적으로 힙이라는 트리 구조를 사용하여 구현하는 것이 가장 일반적이며 효율적이다.


* 특징: 일반적인 큐가 선입선출 구조를 따르는 것과 달리, 각 요소에 우선순위 값이 부여되어 가장 높은 우선순위를 가진 요소가 먼저 처리된다.
* 주요 연산: 데이터를 넣고 우선순위에 맞게 재배치하는 연산을 push, 가장 우선순위가 높은 데이터를 꺼내고 트리를 재구성하는 연산을 pop이라고 한다. 가장 높은 우선순위 데이터를 확인만 하는 연산은 peek이다.
* 장점: 데이터의 삽입 순서와 무관하게 항상 가장 중요하거나 기준에 부합하는 데이터를 빠르게 추출하여 처리할 수 있다. 작업 스케줄링이나 다익스트라 같은 최단 경로 탐색 알고리즘에 매우 유용하다.
* 단점: 일반적인 선형 자료구조에 비해 구현이 복잡하다. 삽입과 삭제 시 내부적으로 트리 구조를 유지하기 위한 정렬 과정이 필요하므로 스택이나 기본 큐보다 연산 시간이 더 소요된다.

## 2. 의사코드

우선순위 큐를 값이 클수록 우선순위가 높은 최대 힙 배열로 구현했을 때의 핵심 연산에 대한 기본 논리를 보여주는 의사코드다.

```text
// 1. 데이터 삽입
function push(priority_queue, value, size):
    size = size + 1
    priority_queue[size] = value
    current = size
    
    // 부모 노드와 비교하며 위로 올라가며 자리 변경
    while current > 1 AND priority_queue[current] > priority_queue[current / 2]:
        swap(priority_queue[current], priority_queue[current / 2])
        current = current / 2

// 2. 데이터 삭제 및 반환
function pop(priority_queue, size):
    if size == 0:
        return "큐 비어있음 오류"
    
    highest_priority_value = priority_queue[1]
    priority_queue[1] = priority_queue[size]
    priority_queue[size] = null
    size = size - 1
    
    current = 1
    // 자식 노드와 비교하며 아래로 내려가며 자리 변경
    while current * 2 <= size:
        largest = current * 2
        if largest + 1 <= size AND priority_queue[largest + 1] > priority_queue[largest]:
            largest = largest + 1
            
        if priority_queue[current] >= priority_queue[largest]:
            break
            
        swap(priority_queue[current], priority_queue[largest])
        current = largest
        
    return highest_priority_value

// 3. 우선순위가 가장 높은 데이터 확인
function peek(priority_queue, size):
    if size == 0:
        return "큐 비어있음 오류"
    
    return priority_queue[1]
```

## 3. 시간 복잡도

우선순위 큐는 힙 트리 구조를 기반으로 하므로 데이터의 이동이 트리의 높이 안에서 이루어진다.

| 연산 | 시간 복잡도 | 설명 |
| :--- | :--- | :--- |
| 삽입 | O(log n) | 트리의 맨 끝에 데이터를 추가한 후 부모 노드와 비교하며 자리를 교환하므로 트리의 높이만큼 시간이 소요된다. |
| 삭제 | O(log n) | 루트 노드를 제거하고 마지막 노드를 루트로 옮긴 후 자식 노드와 비교하며 자리를 교환하므로 트리의 높이만큼 시간이 소요된다. |
| 확인 | O(1) | 루트 노드의 위치를 항상 알고 있으므로 즉시 접근하여 확인한다. |
| 검색 | O(n) | 특정 데이터를 찾기 위해서는 트리 전체 노드를 탐색해야 한다. |

## 4. C 언어 및 파이썬 예시

### C 언어

C 언어에서는 힙 구조를 직접 1차원 배열로 구현하여 최대 힙 기반의 우선순위 큐를 만든다.

```c
#include <stdio.h>
#define MAX_SIZE 100

int priority_queue[MAX_SIZE];
int size = 0;

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

// 데이터 삽입
void push(int value) {
    if (size == MAX_SIZE - 1) {
        printf("큐 공간 가득 참\n");
        return;
    }
    
    priority_queue[++size] = value;
    int current = size;
    
    while (current > 1 && priority_queue[current] > priority_queue[current / 2]) {
        swap(&priority_queue[current], &priority_queue[current / 2]);
        current /= 2;
    }
}

// 데이터 삭제 및 반환
int pop() {
    if (size == 0) {
        printf("큐 비어있음\n");
        return -1;
    }
    
    int highest = priority_queue[1];
    priority_queue[1] = priority_queue[size--];
    
    int current = 1;
    while (current * 2 <= size) {
        int largest = current * 2;
        if (largest + 1 <= size && priority_queue[largest + 1] > priority_queue[largest]) {
            largest++;
        }
        if (priority_queue[current] >= priority_queue[largest]) {
            break;
        }
        swap(&priority_queue[current], &priority_queue[largest]);
        current = largest;
    }
    return highest;
}

int main() {
    push(10);
    push(30);
    push(20);

    printf("pop된 값: %d\n", pop());
    printf("pop된 값: %d\n", pop());

    return 0;
}
```

### 파이썬

파이썬에서는 내장 모듈인 heapq를 사용하여 우선순위 큐를 매우 쉽게 구현할 수 있다. 기본적으로 값이 작을수록 우선순위가 높은 최소 힙으로 동작한다.

```python
import heapq

# 큐 초기화
priority_queue = []

# 데이터 삽입
heapq.heappush(priority_queue, 30) # push 연산
heapq.heappush(priority_queue, 10)
heapq.heappush(priority_queue, 20)

# 가장 높은 우선순위 데이터 확인
if priority_queue:
    print(f"가장 높은 우선순위 값: {priority_queue[0]}") # peek 연산

# 데이터 삭제 및 반환
print(f"pop된 값: {heapq.heappop(priority_queue)}") # pop 연산
print(f"pop된 값: {heapq.heappop(priority_queue)}")

# 현재 큐 상태 확인
print(f"남은 큐: {priority_queue}")
```
