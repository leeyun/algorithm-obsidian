## 1. 스택에 대한 설명

스택은 데이터의 삽입과 삭제가 저장소의 맨 위 한쪽 끝에서만 일어나는 선형 자료구조다. 가장 나중에 저장된 데이터가 가장 먼저 나오는 후입선출(LIFO) 구조를 가진다.

* **후입선출 구조:** 데이터를 쌓아 올리는 형태로 저장하며, 마지막에 추가된 요소가 가장 먼저 제거된다.
* **주요 연산:** 데이터를 넣는 연산을 푸시, 데이터를 꺼내는 연산을 팝이라고 하며, 맨 위의 데이터를 확인만 하는 연산을 피크라고 한다.
* **장점:** 데이터의 삽입과 삭제가 항상 한쪽 끝에서만 발생하므로 구현이 매우 쉽고 속도가 빠르다.
* **단점:** 중간에 있는 데이터에 직접 접근할 수 없다. 인덱스 접근이 불가능하므로 원하는 데이터를 찾을 때까지 위에서부터 차례대로 꺼내야 한다. 또한 크기가 제한된 배열로 구현할 경우 저장 공간이 가득 차면 더 이상 데이터를 넣을 수 없는 스택 오버플로우가 발생한다.

## 2. 의사코드

스택의 핵심 연산인 push(), pop(), peek()에 대한 기본 논리를 보여주는 의사코드다.

```c
// 1. 데이터 삽입 (Push)
function push(stack, value, top, max_size):
    if top == max_size - 1:
        return "스택 오버플로우 오류"
    
    top = top + 1
    stack[top] = value

// 2. 데이터 삭제 및 반환 (Pop)
function pop(stack, top):
    if top == -1:
        return "스택 언더플로우 오류"
    
    value = stack[top]
    stack[top] = null
    top = top - 1
    return value

// 3. 맨 위 데이터 확인 (Peek)
function peek(stack, top):
    if top == -1:
        return "스택이 비어있음"
    
    return stack[top]
```

## 3. 시간 복잡도

스택은 모든 삽입과 삭제가 맨 위에서만 이루어지기 때문에 이 두 연산에 대해 매우 효율적이다.

| 연산 | 시간 복잡도 | 설명 |
| :--- | :--- | :--- |
| 삽입 | O(1) | 맨 위에 데이터를 바로 추가하므로 다른 요소를 이동시킬 필요가 없다. |
| 삭제 | O(1) | 맨 위의 데이터를 바로 제거하므로 다른 요소를 이동시킬 필요가 없다. |
| 확인 | O(1) | 맨 위 요소의 위치를 항상 알고 있으므로 즉시 접근하여 확인한다. |
| 검색 | O(n) | 특정 데이터를 찾기 위해 맨 위부터 원하는 값이 나올 때까지 하나씩 꺼내며 탐색해야 한다. |

## 4. C  및 Python 예시

### C

C 언어에서는 배열과 변수 하나를 활용하여 고정된 크기의 스택을 구현한다.

```c
#include <stdio.h>
#define MAX_SIZE 5

int stack[MAX_SIZE];
int top = -1;

// 데이터 삽입
void push(int value) {
    if (top == MAX_SIZE - 1) {
        printf("스택 오버플로우\n");
        return;
    }
    stack[++top] = value;
}

// 데이터 삭제 및 반환
int pop() {
    if (top == -1) {
        printf("스택 언더플로우\n");
        return -1;
    }
    return stack[top--];
}

int main() {
    push(10);
    push(20);
    push(30);

    printf("팝된 값: %d\n", pop());
    printf("팝된 값: %d\n", pop());

    return 0;
}
```

### Python

파이썬의 리스트는 내장 메서드를 통해 스택의 기능을 기본적으로 제공한다.

```python
# 스택 초기화
stack = []

# 데이터 삽입 (Push)
stack.append(10)
stack.append(20)
stack.append(30)

# 맨 위 데이터 확인 (Peek)
if stack:
    print(f"맨 위의 값: {stack[-1]}")

# 데이터 삭제 및 반환 (Pop)
print(f"팝된 값: {stack.pop()}")
print(f"팝된 값: {stack.pop()}")

# 현재 스택 상태 확인
print(f"남은 스택: {stack}")
```
