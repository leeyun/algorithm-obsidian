## 1. 배열에 대한 설명

배열은 프로그래밍에서 가장 기본적이고 널리 사용되는 선형 자료구조다.

* 연속적인 메모리 할당: 동일한 자료형을 가진 데이터가 메모리상에 연속적으로 나열되어 저장된다.
* 인덱스 접근: 각 데이터는 위치를 나타내는 인덱스를 가지며, 이를 통해 특정 위치의 데이터에 즉각적으로 접근할 수 있다. 인덱스는 보통 0부터 시작한다.
* 고정된 크기: 정적 배열은 선언 시점에 크기가 결정되며 이후에는 크기를 변경할 수 없다. 파이썬의 리스트 같은 동적 배열은 필요에 따라 내부적으로 크기를 자동으로 조절한다.
* 장점: 인덱스를 통한 접근 속도가 매우 빠르다.
* 단점: 크기가 고정되어 있어 메모리 낭비가 발생할 수 있다. 배열 중간에 데이터를 삽입하거나 삭제할 때 나머지 요소를 모두 이동시켜야 하므로 비효율적이다.

## 2. 의사코드

배열의 핵심 연산인 접근, 삽입, 삭제의 의사코드이다.

```text
// 1. 요소 접근
function getElement(array, index):
    if index < 0 OR index >= array.length:
        return "인덱스 범위 초과 오류"
    return array[index]

// 2. 요소 삽입
function insertElement(array, index, value, current_elements_count):
    if current_elements_count == array.length:
        return "배열 공간 부족 오류"
    if index < 0 OR index > current_elements_count:
        return "인덱스 범위 초과 오류"
    
    // 삽입할 위치부터 끝까지의 요소를 한 칸씩 뒤로 이동시킨다.
    for i from current_elements_count down to index + 1:
        array[i] = array[i - 1]
    
    array[index] = value
    current_elements_count = current_elements_count + 1

// 3. 요소 삭제
function deleteElement(array, index, current_elements_count):
    if current_elements_count == 0:
        return "배열 비어있음 오류"
    if index < 0 OR index >= current_elements_count:
        return "인덱스 범위 초과 오류"
    
    // 삭제할 위치 다음부터 끝까지의 요소를 한 칸씩 앞으로 이동시킨다.
    for i from index to current_elements_count - 2:
        array[i] = array[i + 1]
    
    array[current_elements_count - 1] = null
    current_elements_count = current_elements_count - 1
```

## 3. 시간 복잡도

배열은 특정 위치에 접근할 때 압도적으로 유리하지만, 데이터 삽입 및 삭제 시에는 요소를 이동시켜야 하므로 연산 비용이 발생한다.

| 연산 | 시간 복잡도 | 설명 |
| :--- | :--- | :--- |
| 접근 | O(1) | 인덱스를 통해 메모리 주소를 즉시 계산하여 접근한다. |
| 검색 | O(n) | 특정 값을 찾기 위해 처음부터 끝까지 순차적으로 탐색한다. |
| 삽입 | O(n) | 맨 끝에 추가할 때는 O(1)이지만, 중간이나 맨 앞에 삽입할 경우 기존 요소를 뒤로 밀어야 하므로 최악의 경우 O(n)이 소요된다. |
| 삭제 | O(n) | 맨 끝 요소를 제거할 때는 O(1)이지만, 중간이나 맨 앞의 요소를 삭제할 경우 빈자리를 채우기 위해 요소를 앞으로 당겨야 하므로 최악의 경우 O(n)이 소요된다. |

## 4. C 및 Python 예시

### C

C 언어의 배열은 크기를 미리 지정해야 하는 전통적인 정적 배열이다.

```c
#include <stdio.h>

int main() {
    // 1. 배열 선언 및 초기화
    int arr[5] = {10, 20, 30, 40, 50};
    
    // 2. 인덱스를 통한 요소 접근
    printf("인덱스 2의 값: %d\n", arr[2]);
    
    // 3. 값 변경
    arr[2] = 99;
    
    // 4. 배열 순회
    printf("배열 요소: ");
    for(int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }
    
    return 0;
}
```

### Python

파이썬의 리스트는 내부적으로 동적 배열로 구현되어 있어 크기를 자유롭게 변경할 수 있다.

```python
# 1. 배열 선언 및 초기화
arr = [10, 20, 30, 40, 50]

# 2. 인덱스를 통한 요소 접근
print(f"인덱스 2의 값: {arr[2]}")

# 3. 배열 맨 끝에 요소 추가
arr.append(60)

# 4. 특정 인덱스에 요소 삽입
arr.insert(2, 99)
print(f"삽입 후 배열: {arr}")

# 5. 특정 인덱스의 요소 삭제
deleted_value = arr.pop(2)
print(f"삭제된 값: {deleted_value}, 삭제 후 배열: {arr}")
```
